1. Aller sur https://business.google.com
2. Créer une fiche avec :

   - **Catégorie** : Concepteur de sites Web
   - **Zone de service** : Lyon, Auvergne-Rhône-Alpes (pas d'adresse physique nécessaire)
   - **Site web** : https://atlamaz-studio.vercel.app (à remplacer par atlamazstudio.fr dès que le domaine est actif)
   - **Téléphone** : [ton numéro — mettre à jour dans .env VITE_PHONE]

     > "Studio de création web freelance à Lyon. Sites vitrines, landing pages, e-commerce et applications mobiles React Native. Livraison en 5–14 jours. Devis gratuit sous 24h."

3. Vérifier la fiche (carte postale ou appel téléphonique)

4. Ajouter des photos :
   - Logo Atlamaz Studio (format carré, fond noir)
   - Captures d'écran des 3 réalisations (L'Attache, Cabinet Moreau, Maïa Créations)
   - Photo de profil professionnelle

5. Compléter les horaires (ex : Lun–Ven 9h–19h)

6. Ajouter le lien Calendly dans "Prendre RDV" (https://calendly.com/atlamazstudio/30min)

## Après vérification

- Demander un avis à chaque client livré (envoyer le lien direct Google Review)
- Lien direct vers la page avis : `https://g.page/r/[TON_ID]/review`
- **Objectif** : 3 avis 5 étoiles dans le premier mois = apparition dans le "pack local" Google Maps
